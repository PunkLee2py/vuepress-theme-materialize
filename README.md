## Introduce

-   This is a Vuepress theme
-   Themes built with [Materialize UI](https://materializecss.com/)
-   Designed to provide the most convenient experience

## Quick start

**npm**

```bash
npm install vuepress-theme-materialize -D
```

**vuepress**

```
// .vuepress/config.js
module.exports = {
  theme: "materialize"
}
```
